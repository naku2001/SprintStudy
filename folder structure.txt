## Startup Quick Path

1. Copy env template

```powershell
Copy-Item .env.example .env
```

2. Fill required keys in `.env`

- `GEMINI_API_KEY`
- `PINECONE_API_KEY`
- Optional: `TOGETHER_API_KEY` (only needed if you choose `openai/gpt-oss-120b` in UI)
- Optional: `HUGGINGFACE_API_KEY` (only needed if you choose `BAAI/bge-small-zh` embedding)

3. Start backend

```powershell
pip install -r backend\requirements.txt
python backend\main.py
```

4. Start frontend in a new terminal

```powershell
cd frontend
npm install
npm run dev
```

5. Open browser

- Frontend: `http://localhost:3000`
- Backend health: `http://localhost:8000/health`

## Notes

- Library supports: `Re-summarize`, `Open Summary`, `Rename`, `View Chunks`, `Delete`.
- Saved summary is persisted per note and overwritten on re-summarize.
- Vite dev server proxies `/api/*` to Flask (`localhost:8000`).
